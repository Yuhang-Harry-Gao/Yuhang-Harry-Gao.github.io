  • 对于假设矩阵 $\mathbf{A}$ 和矩阵 $\mathbf{B}$ 都是可逆的，怎么求矩阵 $\mathbf{A}\mathbf{B}$ 的逆？

    先穿袜子，后穿鞋。那就先脱鞋，再脱袜子: $$(\mathbf{A}\mathbf{B})(\mathbf{B^{-1}}\mathbf{A^{-1}})=\mathbf{I}$$ $$(\mathbf{B}\mathbf{A})(\mathbf{A^{-1}}\mathbf{B^{-1}})=\mathbf{I}$$

  • 转置矩阵

    • 转置矩阵的逆 = 逆矩阵的转置 $$\mathbf{(A^{-1})^T} = \mathbf{(A^{T})^{-1}}$$

矩阵A的LU分解

非所有矩阵都能进行$\mathbf{LU}$分解，能够LU分解的矩阵需要满足以下三个条件：

  1. 矩阵是方阵（$\mathbf{LU}$分解主要是针对方阵）；
  2. 矩阵是可逆的，也就是该矩阵是满秩矩阵，每一行都是独立向量；
  3. 消元过程中没有0主元出现，也就是消元过程中不能出现行交换的初等变换。
  • $\mathbf{LU}$ 和 $\mathbf{LDU}$

总运算次数是： $$N = n^2 + (n-1)^2 + (n-2)^2 + \cdots + 2^2 + 1^2 $$

$$\text{set } \Delta n = 1$$ $$N \approx \int_1^n n^2 dn \approx \frac{1}{3}n^3$$

转置，置换，向量空间

置换矩阵P（Permutation Matrix）

对于 $n \times n$ 矩阵，一共有 $n!$ 个不同的置换矩阵

转置矩阵（Transpose Matrix）

特殊情况：Symmetric Matrices

  • $\mathbf{A}^T = \mathbf{A}$
  • $\mathbf{R}^T \mathbf{R}$ is always symmetric matrices
    • $(\mathbf{R}^T \mathbf{R})^T = \mathbf{R}^T \mathbf{R}^{TT}= \mathbf{R}^T \mathbf{R}$

对比

对比项 置换矩阵P（Permutation Matrix） 转置矩阵T（Transpose Matrix）
定义 通过交换单位矩阵的行或列构成的矩阵 将矩阵的行列互换
功能 用于表示交换行或列的操作 改变矩阵结构，用于转置操作
作用举例 $\mathbf{PA}$ 表示交换矩阵 $\mathbf{A}$ 的行 $\mathbf{A}^T$ 把第 i 行变成第 i 列，对于 $\mathbf{A}^T_{ij}$ 变成$\mathbf{A}^T _{ji}$
元素结构 只包含 0 和 1，每行每列仅一个 1，其余为0 与原矩阵数值相同，只是行列交换
是否可逆 总是可逆，且 $\mathbf{P}^{-1} = \mathbf{P}^T$ 若 $\mathbf{A}$ 可逆，则 $(\mathbf{A}^T)^{-1} = (\mathbf{A}^{-1})^T$
常见用途 $\mathbf{LU}$分解换行($\mathbf{PA} = \mathbf{LU}$)、组合优化、排序 向量转置、矩阵导数、对称性分析
