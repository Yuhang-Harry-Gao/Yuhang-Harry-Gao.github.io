贝叶斯公式的简单介绍

贝叶斯公式

1.回顾条件概率

条件概率 $P(A|B)$ 回答了这样一个问题： “在事件 B 已经发生的前提下，事件 A 发生的概率是多少？”

$$P(A|B) = \frac{P(A \text{ and } B)}{P(B)}$$

这个公式是说，A在B发生的条件下发生的概率，等于A和B同时发生的概率，除以B自身发生的概率。
